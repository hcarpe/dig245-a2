How to View the HTML Source Code on a Google Chrome Website

  1. Go to the Google Chrome webpage you want to view the HTML source code of.
  2. Right click anywhere on the webpage.
  3. Towards the end of the drop-down menu, click Inspect.
  4. The HTML source code will be on the right side of your screen!
  5. If you want to see the HTML source code of a specific part of the webpage, highlight that part of the webpage before inpsecting the page. The highlighted component (the element) should also be highlighted in the HTML source code.
